# TikTokViewPager
#### 📱应用简介Synopsis
仿TikTok滑动播放视频Demo实例,视频链接是高速链接播放则更流畅。   
Imitate TikTok sliding to play video demo instances, and if the video link is high-speed, the playback will be smoother.
